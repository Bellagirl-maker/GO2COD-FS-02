# 📖Online Store <a name="about-project"></a>

**[Online Store]** is a project of creating a Online store using the MERN stack framework and store cart data with local storage.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- MongoDB
- Express
- React
- Node.js
- Tailwind Css



### Key Features <a name="key-features"></a>

- Display products with images descriptions, and prices.
- Has shopping cart functionality to add, remove, and view items.
- Show the total price of items in the cart.
